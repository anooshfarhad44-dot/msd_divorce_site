## How to Replace Images

1. Create your image in your preferred design tool (Figma, Canva, Photoshop, etc.)
2. Export the image as **WebP format**
3. Replace the corresponding file in the `public/images/` directory
4. Keep the same filename for everything to work correctly

## Image Tips

- Use WebP format for smaller file sizes
- Aim for file sizes under 200KB per image
- Use consistent aspect ratios:
  - Hero: 4:3
  - Services: 4:3
  - Blog: 16:9
  - Testimonials/Team: 1:1 (square)
- Compress images before uploading
